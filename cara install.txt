Cara Install Proyek:

1. Pastikan Anda sudah menginstal Python (versi 3.6.13).
   Anda dapat mengunduhnya di https://www.python.org/downloads/.

2. Pastikan Anda sudah menginstal pip, yang merupakan package manager untuk Python.
   Jika pip belum terinstal, Anda dapat mengikuti petunjuk di https://pip.pypa.io/en/stable/installation/.

3. Download atau clone proyek ini ke komputer Anda.

4. Navigasikan ke folder proyek yang telah di-download atau di-clone menggunakan terminal/command prompt.

5. Lakukan perintah berikut: pip install -r reqirements.txt

6. Jalankan dengan perintah berikut: python app.py
